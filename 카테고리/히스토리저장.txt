=== 내 보관함 ===

[📋 최근 결과] [⭐ 즐겨찾기] [📂 프로젝트] [🗑 휴지통]

=== 자동저장 시스템 ===

모든 생성 결과 자동저장:
- 생성 완료 시 자동으로 최근 결과에 저장
- 저장 성공 시 우측 하단에 토스트 메시지:
  "✅ 보관함에 자동저장 됐어요!"

저장 데이터 구조 (localStorage):
history_list: 최근 20개 결과
- id (고유번호)
- 생성일시
- 카테고리 (대본/번역/블로그 등)
- 제목 (첫 줄 자동추출)
- 내용 전체
- 언어 (한국어/일본어/동시)
- 글자수
- 즐겨찾기 여부
- 프로젝트 ID

favorites_list: 즐겨찾기 목록

projects_list: 프로젝트 목록
- id
- 이름
- 이모지
- 색상
- 생성일
- 포함 결과 ID 목록

trash_list: 휴지통 (7일 보관)

=== 내보내기/가져오기 ===

전체 보관함 내보내기:
[💾 전체 내보내기] 버튼
→ JSON 파일로 다운로드
→ 다른 기기에서 불러오기 가능

선택 내보내기:
결과 카드 체크박스 선택
→ 선택한 것만 내보내기
→ TXT/JSON 선택

가져오기:
[📥 내보낸 파일 불러오기] 버튼
→ JSON 파일 선택
→ 기존 데이터와 병합

=== 보관함 통계 ===

보관함 상단에 통계 표시:
┌──────────┐ ┌──────────┐ ┌──────────┐
│ 📋       │ │ ⭐       │ │ 📂       │
│ 총 20개  │ │ 즐겨찾기 │ │ 프로젝트 │
│ 결과물   │ │ 5개      │ │ 3개      │
└──────────┘ └──────────┘ └──────────┘

용량 사용:
████████░░ 80% (localStorage 5MB 중 4MB)
"용량이 부족하면 오래된 것부터 삭제해요"

=== 생성 화면 연동 ===

모든 생성 결과 아래에 버튼 추가:
[⭐ 즐겨찾기] [📂 프로젝트에 저장] [💾 저장]
자동저장 토스트 메시지 표시

보관함에서 다시 불러오기:
[✏️ 이 설정으로 다시 만들기] 클릭 시
→ 해당 카테고리로 이동
→ 이전 설정값 자동입력
→ 바로 수정해서 재생성 가능

- 모바일 반응형
