기존 어린이 모드 기능은 절대 건드리지 말고
아래 내용만 추가해줘.

=== 추가할 카테고리: 🎨 캐릭터 놀이터 ===

--- A) 캐릭터 스타일 선택 ---

"어떤 느낌의 캐릭터로 만들까요?"

스타일 카드 (크고 귀엽게):

┌──────────────────────────────┐
│ 🐱 귀요미 고양이 친구        │
│ 동그랗고 귀여운 고양이 스타일│
│ (산리오 느낌)                │
│ [이 스타일로 만들기]         │
└──────────────────────────────┘

┌──────────────────────────────┐
│ 👾 깜찍한 몬스터 친구        │
│ 귀엽고 신비로운 몬스터 스타일│
│ (포켓몬 느낌)                │
│ [이 스타일로 만들기]         │
└──────────────────────────────┘

┌──────────────────────────────┐
│ 🧸 장난꾸러기 인형 친구      │
│ 뾰족귀 귀여운 인형 스타일   │
│ (라부부 느낌)                │
│ [이 스타일로 만들기]         │
└──────────────────────────────┘

┌──────────────────────────────┐
│ 🌟 나만의 캐릭터 만들기!    │
│ 세상에 하나뿐인 내 캐릭터   │
│ [직접 만들기]                │
└──────────────────────────────┘

--- B) 스타일별 기능들 ---

각 스타일 선택 시 아래 기능 제공:

[귀요미 고양이 친구] 선택 시:

1. 🐱 내 고양이 캐릭터 만들기
   색깔 선택: [분홍] [흰색] [검정] [갈색] [직접입력]
   성격 선택: [수줍음] [활발함] [먹보] [잠꾸러기]
   특징: [리본달기] [안경쓰기] [모자쓰기]
   → 내 캐릭터 소개서 자동생성
   → 이미지 프롬프트 자동생성
   (저작권 안전: "original cute cat character,
   kawaii style, pastel colors"로 생성)

2. 🐱 내 고양이 캐릭터 이야기
   → 내 캐릭터가 주인공인 동화 자동생성

3. 🐱 고양이 캐릭터 대화 만들기
   → 내 캐릭터와 AI가 대화하는 이야기

4. 🐱 고양이 캐릭터 일기
   → 내 캐릭터가 오늘 있었던 일을 일기로

5. 🐱 고양이 캐릭터 굿즈 문구
   → 스티커/엽서에 쓸 귀여운 문구 생성

[깜찍한 몬스터 친구] 선택 시:

1. 👾 내 몬스터 캐릭터 만들기
   타입: [불꽃] [물] [풀] [번개] [얼음] [직접입력]
   능력: [달리기빠름] [힘이셈] [날수있음] [마법사용]
   외모: [뿔달림] [꼬리있음] [날개있음] [귀엽고작음]
   → 내 몬스터 캐릭터 카드 자동생성
   → 이미지 프롬프트 자동생성
   (저작권 안전: "original cute monster character,
   fantasy style, colorful"로 생성)

2. 👾 내 몬스터 배틀 이야기
   → 내 몬스터가 주인공인 배틀 이야기

3. 👾 몬스터 도감 만들기
   → 내가 만든 몬스터들 도감 자동생성

4. 👾 몬스터 진화 이야기
   → 내 몬스터가 진화하는 이야기

[장난꾸러기 인형 친구] 선택 시:

1. 🧸 내 인형 캐릭터 만들기
   모양: [뾰족귀] [둥근귀] [토끼귀] [곰귀]
   표정: [장난꾸러기] [무표정쿨함] [해맑음] [졸림]
   색깔: [파랑] [분홍] [초록] [보라] [직접입력]
   → 내 인형 캐릭터 소개서 자동생성
   → 이미지 프롬프트 자동생성
   (저작권 안전: "original vinyl art toy character,
   designer toy style, cute"로 생성)

2. 🧸 내 인형의 하루 이야기
3. 🧸 인형 친구들 파티 이야기
4. 🧸 인형 수집 도감 만들기

--- C) 나만의 캐릭터 만들기 (핵심!) ---

"세상에 하나뿐인 내 캐릭터를 만들어요!"

STEP 1: 기본 정보
이름: [입력창] 예: 뽀짝이
동물/종류: [고양이] [강아지] [토끼] [용] [직접입력]
나이: [아기] [어린이] [어른]

STEP 2: 외모 꾸미기
색깔: 색상 팔레트 선택
크기: [작은편] [보통] [큰편]
특징: [귀가큼] [꼬리있음] [날개있음]
      [뿔있음] [안경씀] [모자씀]
      [리본달림] [직접입력]

STEP 3: 성격 선택 (여러개 가능)
[활발함] [수줍음] [장난꾸러기]
[먹보] [잠꾸러기] [용감함]
[친절함] [웃긴] [직접입력]

STEP 4: 특별 능력
[하늘날기] [빨리달리기] [마법사용]
[음식만들기] [노래잘함] [직접입력]

STEP 5: 좋아하는 것
[입력창] 예: 딸기케이크, 낮잠, 친구들

[✨ 내 캐릭터 완성!] 버튼

자동 생성되는 것들:
1. 📋 캐릭터 소개서
   "안녕! 나는 뽀짝이야!
    나는 분홍 고양이이고
    장난꾸러기에 먹보야!
    특기는 하늘날기고
    딸기케이크를 제일 좋아해!"

2. 🎨 이미지 프롬프트
   (DALL-E/Midjourney에 붙여넣기용)
   "Original cute pink cat character named
   Ppojjaki, has big ears and ribbon,
   playful expression, kawaii style,
   pastel background, child-friendly..."
   [복사] [DALL-E 열기↗]

3. 📖 내 캐릭터 동화
   → 내 캐릭터가 주인공인 짧은 동화

4. 🎤 내 캐릭터 소개 노래 가사
   → 내 캐릭터 테마송 가사 자동생성

5. 🃏 캐릭터 카드
   → 포켓몬 카드처럼 능력치 카드 자동생성
   HP: 85 / 공격: 70 / 귀여움: 999

캐릭터 저장소:
만든 캐릭터 최대 10개 저장 (localStorage)
저장된 캐릭터 카드 목록으로 표시
캐릭터 클릭 → 해당 캐릭터로 이야기/동화 만들기

--- D) 캐릭터 친구들 이야기 ---

저장된 캐릭터 2개 선택
→ 둘이 함께하는 이야기 자동생성
→ 티키타카 대화 자동생성
→ 함께하는 모험 이야기

--- E) 캐릭터 굿즈 문구 생성 ---

"내 캐릭터로 굿즈를 만들어요!"

종류 선택:
[스티커 문구] [엽서 문구]
[머그컵 문구] [파우치 문구]

→ 캐릭터 특징 기반 귀여운 문구 자동생성
→ 한국어/일본어/영어 동시생성

공통:
- 무지개색상 밝고 귀여운 UI
- 크고 귀여운 이모지 풍부하게
- 각 단계마다 [❓ 이게 뭔가요?] 버튼
- 결과 복사/저장/프린트 버튼
- localStorage에 캐릭터 저장
- 어린이 안전: 외부링크 차단
- 모바일 반응형
