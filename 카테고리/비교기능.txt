index.html에 A/B 비교 기능을 추가해줘.
사이드바에 ⚖️ A/B 비교 메뉴를 추가하고
클릭 시 중앙에 아래 화면이 나오게 해줘.

=== A/B 비교 센터 ===

제목: ⚖️ A/B 비교하기
부제: "같은 주제로 2개 만들어서 더 좋은 것 골라요!"

--- 섹션1: 뭘 비교할까요? ---

비교 유형 카드:

┌─────────────────────────────────┐
│ 📝 대본 A/B 비교                │
│ 같은 주제로 스타일이 다른       │
│ 대본 2개 동시생성               │
│ [선택하기]                      │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 🎯 제목 A/B 비교                │
│ 같은 영상에 어울리는            │
│ 제목 5개 중 최고 선택           │
│ [선택하기]                      │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 🖼 썸네일 문구 A/B 비교         │
│ 클릭률 높은 썸네일 문구         │
│ 2가지 버전 비교                 │
│ [선택하기]                      │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 🌏 한국어 vs 일본어 비교        │
│ 같은 내용의 한일 버전           │
│ 나란히 비교/수정                │
│ [선택하기]                      │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 😊 말투 A/B 비교                │
│ 같은 내용을 다른 말투로         │
│ 친근체 vs 전문체 비교           │
│ [선택하기]                      │
└─────────────────────────────────┘

--- 섹션2: 대본 A/B 비교 ---

STEP1: 기본 정보 입력

주제:
┌─────────────────────────────────┐
│ 예: "치매 예방하는 음식 TOP5"   │
└─────────────────────────────────┘

A버전 스타일:
[감동형] [정보형] [스토리형] [퀴즈형] [직접입력]

B버전 스타일:
[감동형] [정보형] [스토리형] [퀴즈형] [직접입력]

언어: ● 한국어  ○ 일본어  ○ 동시
길이: [30초] [●60초] [3분] [5분]
채널: [시니어감동] [코믹] [지식] [직접입력]

[⚡ A/B 동시 생성!] 큰 버튼

STEP2: 나란히 비교

생성 완료 후 좌우 분할 화면:

┌──────────────────┬──────────────────┐
│   📝 A버전       │   📝 B버전       │
│   감동형 스타일  │   정보형 스타일  │
│                  │                  │
│ 안녕하세요!      │ 오늘 알려드릴    │
│ 오늘은 제 어머니 │ 내용은 의학적으로│
│ 이야기를 해볼게요│ 검증된 치매 예방 │
│ 어머니가 매일    │ 음식 5가지입니다 │
│ 召召召召召召召召 │ 召召召召召召召召 │
│                  │                  │
│ 📏 1,240자       │ 📏 980자         │
│ ⏱ 약 4분        │ ⏱ 약 3분        │
│                  │                  │
│ AI 점수: ★★★★☆  │ AI 점수: ★★★☆☆  │
│ 예상 시청완료율  │ 예상 시청완료율  │
│ 높음 📈          │ 보통 📊          │
│                  │                  │
│ [전체보기]       │ [전체보기]       │
│ [복사]           │ [복사]           │
│                  │                  │
│ [✅ A 선택!]     │ [✅ B 선택!]     │
└──────────────────┴──────────────────┘

AI 추천:
┌─────────────────────────────────────┐
│ 🤖 AI 추천: A버전이 더 좋아요!      │
│                                     │
│ 이유:                               │
│ ✅ 감동 스토리로 시청 완료율 높음   │
│ ✅ 시니어 채널에 더 적합            │
│ ✅ 공유 가능성 높음                 │
│                                     │
│ B버전이 더 좋은 경우:               │
│ → 정보성 채널일 때                  │
│ → 빠른 정보 전달이 목적일 때        │
└─────────────────────────────────────┘

선택 후 액션:
A 또는 B 선택 시:
[⭐ 즐겨찾기] [📂 프로젝트 저장]
[⚡ 숏츠로 만들기] [🎬 미디어 패키지]
[🔄 C버전도 만들기]

--- 섹션3: 제목 A/B 비교 ---

주제 입력:
┌─────────────────────────────────┐
│ 영상 주제나 대본 첫 줄 입력     │
└─────────────────────────────────┘

[🎯 제목 5개 동시생성]

생성된 제목들 카드:

┌─────────────────────────────────┐
│ 1번 제목                        │
│ "치매 예방하는 음식 TOP5 (충격)"│
│                                 │
│ AI 클릭률 예측: ★★★★★ 매우높음 │
│ 이유: 숫자+괄호 강조 효과       │
│                                 │
│ [✅ 이 제목 선택] [복사]         │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 2번 제목                        │
│ "이걸 매일 먹으면 치매가 예방?" │
│                                 │
│ AI 클릭률 예측: ★★★★☆ 높음     │
│ 이유: 의문형으로 궁금증 유발    │
│                                 │
│ [✅ 이 제목 선택] [복사]         │
└─────────────────────────────────┘

(5개 카드 동일 형식)

일본어 제목도 동시생성 토글:
☑ 일본어 제목도 같이 보기
日本語タイトルも同時表示

--- 섹션4: 한국어 vs 일본어 비교 ---

"같은 내용이 두 언어로 어떻게 다른지 봐요"

입력:
┌─────────────────────────────────┐
│ 한국어 대본 또는 주제 입력      │
└─────────────────────────────────┘

[🌏 한일 동시 생성]

나란히 비교:
┌──────────────────┬──────────────────┐
│ 🇰🇷 한국어 버전  │ 🇯🇵 일본어 버전  │
│                  │                  │
│ 안녕하세요!      │ こんにちは！      │
│ 오늘은 치매 예방 │ 本日は認知症予防  │
│ 음식에 대해...   │ 食品について...   │
│                  │                  │
│ [복사]           │ [복사]           │
│ [수정하기]       │ [수정하기]       │
└──────────────────┴──────────────────┘

번역 품질 체크:
[🔍 번역이 자연스러운지 확인]
→ 어색한 표현 자동 감지
→ 개선 제안 표시

--- 섹션5: 말투 A/B 비교 ---

주제:
┌─────────────────────────────────┐
│ 예: "치매 예방 음식 소개"       │
└─────────────────────────────────┘

비교할 말투 선택:

A말투: ○친근하게 ●감동적으로 ○전문적으로
B말투: ○친근하게 ○감동적으로 ●전문적으로

[⚡ 두 말투로 동시 생성]

나란히 비교 후 선택

--- 섹션6: 내 A/B 테스트 기록 ---

"어떤 버전이 실제로 잘됐는지 기록해요"

테스트 기록 카드:
┌─────────────────────────────────────┐
│ 📊 치매 예방 음식 A/B 테스트        │
│ 2025년 4월 15일                     │
│                                     │
│ A버전 (감동형): 조회수 15,000       │
│ B버전 (정보형): 조회수 8,000        │
│                                     │
│ 🏆 승자: A버전 (감동형)             │
│ → 다음 영상에도 감동형 사용!        │
└─────────────────────────────────────┘

[+ 테스트 결과 기록하기]

누적 통계:
내 채널에서 잘되는 스타일:
🥇 감동형 (승률 70%)
🥈 스토리형 (승률 60%)
🥉 정보형 (승률 45%)

AI 학습 결과:
"당신 채널은 감동형+스토리형이
가장 잘 맞아요!"
