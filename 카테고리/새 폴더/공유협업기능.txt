index.html에 공유/저장 기능을 추가해줘.

1. 모든 생성 결과 카드 아래에
   공유 버튼 모음을 추가해줘:

┌─────────────────────────────────────┐
│ 📤 공유하기                         │
│                                     │
│ [📋 복사] [💾 저장] [🖨 인쇄]       │
│                                     │
│ [💬 카카오톡] [📱 라인] [🐦 X공유]  │
│                                     │
│ [📄 PDF 저장] [📝 TXT 저장]         │
└─────────────────────────────────────┘

각 버튼 기능:

📋 복사:
→ 클립보드에 복사
→ "복사됐어요!" 토스트 메시지

💾 저장:
→ 보관함에 저장
→ "보관함에 저장됐어요!" 토스트

🖨 인쇄:
→ 브라우저 인쇄 창 열기
→ 인쇄 최적화 CSS 적용

💬 카카오톡 공유:
→ 카카오톡 공유 API 사용
→ 제목+내용 미리보기로 공유
→ 카카오 미설치 시: "내용 복사 후
   카카오톡에 붙여넣기 하세요" 안내

📱 라인 공유:
→ LINE 공유 URL 스킴 사용
→ https://social-plugins.line.me 활용
→ 일본 채널용 (라인 일본 사용자 많음)

🐦 X(트위터) 공유:
→ twitter.com/intent/tweet 사용
→ 내용 요약+링크 자동생성

📄 PDF 저장:
→ window.print() + CSS @media print 활용
→ 깔끔한 PDF 레이아웃 자동적용
→ 제목/날짜/내용 포함
→ 로고 자동포함

📝 TXT 저장:
→ Blob으로 텍스트 파일 다운로드
→ 파일명: 제목_날짜.txt 자동생성

2. 사이드바에 📤 공유센터 메뉴 추가:

=== 공유 센터 ===

제목: 📤 공유 & 내보내기
부제: "만든 것을 팀원과 나눠요!"

--- 섹션1: 빠른 공유 ---

최근 결과 선택:
보관함 최근 20개 중 선택
→ 선택한 결과 공유

공유 방법:
┌─────────────────────────────────┐
│ 💬 카카오톡으로 보내기          │
│ 팀원/지인에게 바로 전송         │
│ [카카오톡 공유]                 │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 📱 라인으로 보내기              │
│ 일본 팀원에게 전송              │
│ [라인 공유]                     │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 📄 PDF로 저장                   │
│ 깔끔하게 정리된 문서로          │
│ [PDF 만들기]                    │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 📝 텍스트 파일로 저장           │
│ 메모장/워드에서 열 수 있어요    │
│ [TXT 저장]                      │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ 🔗 링크 복사                    │
│ 결과를 텍스트 링크로 공유       │
│ (로컬 저장 후 공유용 텍스트)    │
│ [링크 복사]                     │
└─────────────────────────────────┘

=== PDF 인쇄 최적화 CSS ===

@media print 스타일:
- 불필요한 버튼/사이드바 숨김
- 깔끔한 흰 배경
- 폰트 크기 최적화
- 페이지 나눔 자동처리
- 상단에 제목/날짜 자동표시
- 하단에 "통합콘텐츠 생성기" 워터마크

=== 공통 설정 ===
- 모든 공유 버튼 우측 하단 플로팅으로도 표시
- 공유 성공 시 토스트 메시지
- 모바일 반응형
- 카카오/라인 SDK 로드
- localStorage 연동
