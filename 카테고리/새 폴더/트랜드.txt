index.html에 트렌드 분석 기능을 추가해줘.
사이드바에 🔥 트렌드 분석 메뉴를 추가하고
클릭 시 중앙에 아래 화면이 나오게 해줘.

=== 트렌드 분석 센터 ===

제목: 🔥 지금 뭐가 뜨고 있나요?
부제: "트렌드를 알아야 조회수가 올라가요!"

--- 섹션1: 실시간 트렌드 키워드 ---

탭 4개:
[🇰🇷 한국 유튜브] [🇯🇵 일본 유튜브] [🌍 글로벌] [👴 시니어 특화]

한국 유튜브 탭:
카테고리 필터:
[전체] [시니어/건강] [감동] [코믹] [지식] [음악]

트렌드 키워드 카드 (10개):
┌─────────────────────────────────┐
│ 🔥 1위  치매 예방 음식           │
│ 📈 급상승 중                     │
│ 관련 채널: 건강/시니어           │
│ [이 주제로 대본 만들기 →]        │
│ [이 주제로 숏츠 만들기 →]        │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│ 🔥 2위  노후 준비 방법           │
│ 📈 지난주 대비 +230%            │
│ [이 주제로 대본 만들기 →]        │
└─────────────────────────────────┘
(10개 카드 동일 형식)

주의: 실시간 API 연동 전까지는
샘플 트렌드 데이터로 표시하고
"매일 업데이트 예정" 안내 표시

일본 유튜브 탭:
카테고리 필터:
[全体] [シニア/健康] [感動] [エンタメ] [音楽/演歌]

트렌드 키워드 카드 (10개):
┌─────────────────────────────────┐
│ 🔥 1위  老後の生活費             │
│ 📈 急上昇中                     │
│ [このテーマで台本作成 →]         │
│ [ショート動画を作る →]           │
└─────────────────────────────────┘
(10개 동일 형식)

글로벌 탭:
영어권 트렌드 키워드 10개
각각 [한국어 버전 만들기] [일본어 버전 만들기] 버튼

시니어 특화 탭:
한국/일본 시니어 채널 전용 트렌드
건강/추억/가족/노후 카테고리

--- 섹션2: URL 분석 ---

"유튜브 영상 주소를 넣으면 분석해줘요"

입력창:
┌─────────────────────────────────────┐
│ 유튜브 URL을 여기에 붙여넣으세요    │
│ 예: https://youtube.com/watch?v=... │
└─────────────────────────────────────┘
[🔍 분석하기] 버튼

분석 결과 카드:
┌─────────────────────────────────────┐
│ 📊 영상 분석 결과                   │
│                                     │
│ 제목: "치매 예방하는 3가지 습관"   │
│ 예상 조회수: 높음                   │
│                                     │
│ 잘된 이유:                          │
│ ✅ 제목에 숫자 포함 (3가지)         │
│ ✅ 건강 키워드 포함                 │
│ ✅ 궁금증 유발형 제목               │
│                                     │
│ 이 영상의 핵심 키워드:              │
│ [치매예방] [건강습관] [시니어]      │
│                                     │
│ 비슷한 주제로 만들기:               │
│ [🎬 대본 만들기 →]                  │
│ [⚡ 숏츠 만들기 →]                  │
└─────────────────────────────────────┘

--- 섹션3: 경쟁 채널 분석 ---

"잘되는 채널을 분석해서 배워요"

채널 URL 입력:
┌─────────────────────────────────────┐
│ 분석할 채널 URL을 넣어주세요        │
└─────────────────────────────────────┘
[🔍 채널 분석하기]

분석 결과:
┌─────────────────────────────────────┐
│ 📊 채널 분석 결과                   │
│                                     │
│ 이 채널의 인기 비결:                │
│ ✅ 업로드 주기: 주 3회              │
│ ✅ 최적 업로드 시간: 오후 7시       │
│ ✅ 주력 콘텐츠: 건강/치매 예방      │
│ ✅ 썸네일 스타일: 큰 글씨+강렬한색 │
│ ✅ 평균 영상 길이: 8분              │
│                                     │
│ 인기 영상 TOP5 키워드:              │
│ [치매] [건강] [노후] [음식] [운동]  │
│                                     │
│ 차별화 전략 제안:                   │
│ "이 채널이 안 다루는 주제:"        │
│ → 일본 시니어 트렌드               │
│ → 한일 비교 콘텐츠                 │
│ → 추억의 노래 스토리               │
│                                     │
│ [이 전략으로 대본 만들기 →]         │
└─────────────────────────────────────┘

즐겨찾기 채널 등록:
최대 5개 채널 저장
[+ 채널 추가] 버튼
저장된 채널 카드로 표시
localStorage 저장

--- 섹션6: 트렌드 캘린더 ---

"언제 어떤 주제가 뜨는지 미리 알아요"

1월: 신년/새해 다짐/건강검진
2월: 설날/명절/밸런타인
3월: 봄/건강관리/새학기
4월: 벚꽃/봄나들이/건강식품
5월: 어버이날/가정의달/운동
6월: 여름준비/건강음료/여행
7월: 휴가/더위극복/시원한음식
8월: 광복절/추억/역사이야기
9월: 추석/명절음식/가족이야기
10월: 가을/단풍/건강검진시즌
11월: 김장/겨울준비/따뜻한음식
12월: 연말/크리스마스/한해정리

일본 시즌 트렌드도 동시 표시:
1月: お正月/初詣
2月: 節分/バレンタイン
(12개월 동일)

오늘 날짜 기준 자동 표시:

"다음달 미리 준비할 주제"

[이달의 추천 주제로 대본 만들기 →]

=== 연동 기능 ===

모든 섹션에서:
[🎬 대본 만들기 →] 클릭 시
→ engines/script/index.html로 주제 자동전달

[⚡ 숏츠 만들기 →] 클릭 시
→ engines/shorts/index.html로 주제 자동전달

[🎵 미디어 패키지 →] 클릭 시
→ engines/media/index.html로 연동
