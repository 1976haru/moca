완전히 새로 만들어줘.
기존 내용 전부 교체.

핵심 원칙:
- HTML/CSS로 Canva급 템플릿 구현
  (외부 API 없이 브라우저에서 렌더링)
- 글+이미지+애니메이션 한번에
- 1개 콘텐츠 → 모든 플랫폼 자동 변환
- SEO 자동 최적화
- 수익화 연동
