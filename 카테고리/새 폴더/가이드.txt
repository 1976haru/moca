index.html의 가이드 섹션을 완성해줘.

가이드는 초등학생도 이해할 수 있는 언어로
이모지/그림/비유를 풍부하게 사용해서 만들어줘.

탭 구성 (10개):

=== 탭1: 🚀 온보딩 마법사 ===
처음 접속 시 자동실행되는 단계별 마법사:

Step 1: 나는 누구인가요? (카드 선택)
- 🏪 소상공인/자영업자
- 📝 유튜버/크리에이터
- 🏛️ 공무원/공공기관
- 📚 학생/학부모
- 💼 직장인/프리랜서
- 🎮 그냥 궁금해서

Step 2: 주로 뭘 하고 싶나요?
(Step1 선택에 따라 맞춤 옵션 자동변경)

Step 3: 딱 맞는 기능으로 바로 이동!
→ 30초 만에 첫 결과 완성 안내

=== 탭2: ⚡ 3분 빠른 시작 ===
그림+이모지로 5단계 설명:

[1단계] 🔑 "API 키를 넣어요"
- 게임 계정처럼 딱 한 번만!
- 어디서 받나요? 버튼

[2단계] 👆 "원하는 걸 클릭해요"
- 메뉴에서 골라요
- 예: 블로그 글쓰기

[3단계] ✍️ "주제를 입력해요"
- 예시: "제주도 여행 후기"

[4단계] 🎯 "생성 버튼 클릭!"

[5단계] 🎉 "완성! 복사해서 쓰면 끝!"

목적별 3분 가이드 탭:
- 블로그 글 3분 만들기
- 숏츠 대본 3분 만들기
- 공문서 3분 만들기
- 메뉴판 3분 만들기
- 숙제도움 3분 받기

바로 써먹는 예시 모음:
- 카테고리별 예시 결과물 3개씩
- 복사버튼으로 바로 사용

=== 탭3: 📖 초등학생도 아는 개념 설명 ===
카드 형태로 개념 설명:

AI가 뭔가요?
"AI는 엄청 똑똑한 친구예요"
😊 보통 글쓰기: 30분~1시간
🤖 AI 글쓰기: 30초~1분
차이: ⏰ 29분 절약!

API 키가 뭔가요?
"도서관 회원증이에요"
🏛️ 도서관(AI회사)에 회원가입하면
회원증(API키)을 줘요
비용: 글 1개 = 약 10원~50원

프롬프트가 뭔가요?
"AI한테 보내는 편지예요"
❌ 나쁜 편지: "블로그 글 써줘"
✅ 좋은 편지: 
"제주도 2박3일 여행 후기를
30대 여성이 읽기 좋게
친근한 말투로 써줘"

AI 모델이 뭔가요?
"AI 선생님 종류예요"
🟣 Claude = 글쓰기 잘해요
🟢 GPT = 다방면으로 잘해요
🔵 Gemini = 정보검색 잘해요

토큰이 뭔가요?
"AI가 읽는 글자 수예요"
짧게 핵심만 입력하면 비용 절약!

카드 형태로 비교:

✅ After: 3분 완성 / 월 9,900원 무제한
💰 월 최대 100만원 절약!

공무원 Before/After:
❌ Before: 보고서 1개 = 2~3시간 / 야근 반복

⏰ 월 평균 20시간 절약!

유튜버 Before/After:
❌ Before: 대본 1개 = 2~4시간 / 주 2개 올리기 힘듦
✅ After: 30분 완성 / 주 7개 가능

학부모 Before/After:
❌ Before: 숙제 도움 = 막막하고 힘듦

😊 아이와 즐겁게 공부

=== 탭5: 💰 비용 계산기 ===
인터랙티브 계산기 3종:

AI 비용 계산기:
- 월 생성 횟수 슬라이더 (1~500회)
- Claude/OpenAI/Gemini 예상 비용 자동계산
- 가장 저렴한 선택 자동추천

절약 비용 계산기:

  SNS대행 절약: 월 50만원
  전단지 절약: 연 120만원
  합계: 연 최대 XXX만원 절약

→ 예상 광고수익 자동계산

=== 탭8: 🔧 스마트 문제해결 ===
의사결정 트리 방식:

"어떤 문제인가요?" 버튼 선택:
🔴 아예 안 돼요
→ API키 확인→인터넷연결→새로고침

🟡 결과가 이상해요
→ 입력구체적으로→톤길이설정→다시생성

🟠 너무 짧게 나와요
→ 길이설정늘리기→더자세히추가

🟣 한국어가 어색해요
→ Claude모델선택→한국어전용설정

🔵 저장이 안 돼요
→ 복사버튼사용→브라우저허용설정

API 오류 코드 입력창:
오류 코드 입력 → 원인+해결방법 자동안내

그래도 해결 안 될 때:
→ 1:1 문의하기 버튼
→ 화면 캡처 방법 안내
