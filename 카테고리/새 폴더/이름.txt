Q&A 목록 (아코디언 방식 - 클릭하면 열림):

[시작하기]

Q1. MOCA가 뭔가요?
A: MOCA는 "Make One Click All"의 줄임말이에요.
   클릭 하나로 대본, 번역, SNS 게시물, 보고서 등
   모든 콘텐츠를 AI가 자동으로 만들어주는 프로그램이에요.
   마치 마법처럼요! ✨

Q2. 코딩을 몰라도 쓸 수 있나요?
A: 네! 전혀 몰라도 돼요.
   클릭하고 글자 입력하면 끝이에요.
   초등학생도 쓸 수 있어요! 🎮

Q3. API 키가 뭔가요? 어렵지 않나요?
A: API 키는 도서관 회원증 같은 거예요.
   딱 한 번만 만들면 돼요.
   만드는 방법은 가이드 탭에서 쉽게 알려드려요!
   5분이면 완성! ⏱

Q4. 처음에 뭐부터 해야 하나요?
A: 이 순서대로 따라하세요:
   1. 설정 탭 → API 키 입력
   2. 왼쪽 카테고리 중 원하는 것 클릭
   3. 주제 입력
   4. 생성 버튼 클릭!
   끝이에요! 😊

[비용/결제]

Q5. 얼마나 드나요?
A: 두 가지 비용이 있어요.
   1) MOCA 이용료: 월 9,900원~29,900원
   2) AI 사용료: 글 1개당 약 10원~50원
   합쳐도 월 1~3만원이면 충분해요!

Q6. 무료로 써볼 수 있나요?
A: API 키만 있으면 MOCA는 무료로 써볼 수 있어요.
   API 키도 처음엔 무료 크레딧이 있어요!

Q7. API 비용이 걱정돼요
A: 걱정 마세요!
   글 1개 = 약 10원~50원이에요.
   하루 10개 만들어도 월 3,000원~15,000원이에요.
   카페 아메리카노 3잔 가격이에요! ☕

Q8. 어떻게 결제하나요?
A: AI 회사(Anthropic/OpenAI)에
   직접 충전하면 돼요.
   신용카드로 5달러(약 7,000원)부터 충전 가능해요.

[기능]

Q9. 일본어도 나오나요?
A: 네! 한국어와 일본어가 동시에 나와요.
   따로 번역 안 해도 돼요.
   한 번에 두 채널 운영 가능해요! 🇰🇷🇯🇵

Q10. 어떤 AI를 사용하나요?
A: Claude(Anthropic), GPT(OpenAI), Gemini(Google)
   세 가지를 쓸 수 있어요.
   설정에서 원하는 것으로 바꿀 수 있어요!

Q11. 유튜브 자동 업로드도 되나요?
A: 현재는 업로드용 텍스트(제목/설명/태그)를
   자동으로 만들어줘요.
   직접 업로드는 복사해서 붙여넣기 하면 돼요.
   자동 업로드는 추후 업데이트 예정이에요!

Q12. 저장은 어떻게 되나요?
A: 만든 결과물이 자동으로 보관함에 저장돼요.
   최근 20개까지 보관해요.
   즐겨찾기와 프로젝트별 관리도 가능해요! 📁

Q13. 몇 개까지 만들 수 있나요?
A: 무제한이에요!
   API 크레딧이 있는 한 계속 만들 수 있어요.

Q14. 스마트폰에서도 되나요?
A: 네! 모바일에서도 잘 돼요.
   화면이 자동으로 스마트폰 크기에 맞게 바뀌어요! 📱

[오류해결]

Q15. 결과가 안 나와요
A: 이렇게 확인해보세요:
   1) 설정 탭에서 API 키가 입력됐는지 확인
   2) API 크레딧이 있는지 확인
   3) 인터넷 연결 확인
   그래도 안 되면 페이지 새로고침(F5) 해보세요!

Q16. 일본어가 이상해요
A: Claude 모델을 선택하면 일본어 품질이 좋아져요.
   설정 탭에서 Claude로 바꿔보세요!

Q17. 화면이 깨져 보여요
A: 브라우저를 Chrome으로 써보세요.
   또는 페이지를 새로고침(F5) 해보세요!

Q18. 저장이 안 돼요
A: 브라우저의 로컬 저장소를 사용해요.
   시크릿 모드에서는 저장이 안 될 수 있어요.
   일반 모드로 사용해주세요!

[판매/활용]
