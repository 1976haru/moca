engines/media/index.html 파일을 새로 만들어줘.

대상: 다채널 운영자 (한국+일본 유튜브, 숏츠+롱폼+릴스+팟캐스트+광고+교육)

탭 6개 구성:

탭1 - 콘텐츠 입력
- 콘텐츠 유형 선택 카드:
  대본(숏츠) / 대본(롱폼) / 뉴스레터 / 전자책 / 블로그 / 공공기관 / 소상공인 / 교육
- 텍스트 붙여넣기 입력창 (크게)
- 다른 엔진에서 자동 연동 시 콘텐츠 자동 입력
- 언어 감지 자동 (한국어/일본어/영어)
- 콘텐츠 입력 후 유형 자동 분석 버튼

탭2 - 채널/용도 설정
- 채널 프로파일 저장/불러오기
- 용도 선택: 유튜브숏츠 / 유튜브롱폼 / 인스타릴스 / 팟캐스트 / 광고 / 교육
- 언어: 한국어 / 일본어 / 영어
- 장르: 시니어감동 / 코믹 / 티키타카 / 지식 / 드라마 / 광고 / 교육
- 캐릭터: 1인 / 2인티키타카 / 다인
- 채널 프로파일 localStorage 저장

탭3 - 음성 자동매칭
- 콘텐츠+채널 설정 기반 자동 추천
- 장르별 자동 매칭:
  시니어감동 → 따뜻하고 느린 목소리
  코믹캐릭터 → 밝고 과장된 목소리
  티키타카 → 캐릭터A/B 각각 다른 목소리
  지식채널 → 신뢰감 있는 목소리
  드라마 → 감정 풍부한 목소리
  광고 → 에너지 넘치는 목소리
  교육 → 차분하고 명확한 목소리
- 추천 목소리 3개 표시 + 가성비 순 정렬
- TTS 공급자 가성비 자동 선택:
  기본 → OpenAI TTS (저렴)
  고품질 → ElevenLabs (특별편만)
  한국어 → CLOVA Voice
  일본어 → Nijivoice
- 한국어+일본어 음성 설정 동시 출력
- 씬별 다화자 스크립트 자동 구성

탭4 - 이미지 자동생성
- 콘텐츠 유형별 자동 분기:
  숏츠/롱폼 → 씬별 이미지 프롬프트
  뉴스레터 → 헤더 이미지 프롬프트
  전자책 → 챕터 표지 프롬프트
  블로그 → 대표 이미지 프롬프트
  광고 → 상품/서비스 이미지 프롬프트
  교육 → 슬라이드 이미지 프롬프트
- 가성비 자동 선택:
  일반 → DALL-E 3 (저렴)
  썸네일 텍스트 → Ideogram (무료)
  캐릭터 일관성 → Flux (고품질)
- 스타일 선택: 실사 / 일러스트 / 애니메이션 / 인포그래픽
- 한국/일본 채널별 스타일 프리셋
- 이미지 프롬프트 복사 버튼

탭5 - 영상 조립
- 콘텐츠 유형별 자동 분기:
  숏츠 → 이미지+음성 자동 조립 씬 구조 (무료)
  롱폼 → InVideo 씬 프롬프트 생성
  릴스 → 세로형 영상 구조 자동 생성
  광고 → HeyGen 아바타 스크립트 생성
  교육 → 슬라이드+음성 구조 자동 생성
  전자책 → 프로모 영상 씬 자동 생성
- A방식(자동조립)+B방식(InVideo) 병행
- 씬별 타이밍/전환효과 자동 설정
- 자막 위치/폰트 한국어/일본어 자동 설정

탭6 - 원클릭 전체 패키지
- 콘텐츠 붙여넣기만 하면 전체 자동 처리:
  → 콘텐츠 유형 자동 감지
  → 채널 프로파일 자동 적용
  → 음성 설정 자동 생성
  → 이미지 프롬프트 자동 생성
  → 영상 구조 자동 생성
  → 썸네일+제목+태그 자동 생성
  → 전체 패키지 한번에 출력
- 결과 전체 복사/저장 버튼

- 상단 허브로 돌아가기 버튼
